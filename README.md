# drawing
A blank canvas on which one can freely draw.

# Requirements
• Windows 7 SP1 or newer

• .NET Framework 4.7.2 or newer

# Download
Click the Code button, then click on "Download ZIP"

# Execution
Run Drawing.exe, located in Drawing\Drawing\bin\Debug

# How to Use
• Hold left click and drag the cursor on the app to draw.

• Click on "Color" to select the color of your brush. Select one of the default colors, or create a custom one.

• Click on "Line Thickness" to select the thickness of your brush from 7 different options. The fourth option is selected by default.

• Click on "Shape" to select the shape that will be automatically drawn on the app when you click on it. The available shapes are Line, Oval, Circle and Rectangle. The dimensions and the location of each shape are random. The color and the line thickness of each shape are the currently selected ones. The option "No Shape" is selected by default.

• Click the Draw a House button to automatically draw a house line by line. The color and the line thickness of the house will be the currently selected ones.
